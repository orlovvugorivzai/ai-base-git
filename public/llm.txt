# AI-база

Закрита Telegram-спільнота на тему AI. Цей сайт слугує екосистемою та візитівкою для нашої спільноти.

## Основна інформація
- Домен: https://ai-base.one
- Мова: Українська
- Тематика: Штучний інтелект, розробка, програмування, LLM

## Структура сайту:
- `/` - Головна сторінка. Загальна інформація про спільноту.
- `/rules` - Правила спільноти.
- `/networking` - Платформа для нетворкінгу та пошуку однодумців.
- `/github` - Наші open-source проєкти та контрибуції.
- `/guides` - Корисні посібники та гайди.
- `/services` - Внутрішні сервіси та послуги.
- `/hermes` - Інформація про Hermes (наш спеціальний проєкт/інструмент).
- `/pirate-bay` - Корисні ресурси, матеріали та посилання.
- `/shitposting` - Розділ для гумору, мемів та вільного спілкування.
